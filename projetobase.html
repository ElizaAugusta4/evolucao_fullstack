HTML5 e CSS3

HTML5 é composto por tags e atributos, código traduzido pelo navegador. Já CSS3 trata-se do estilo do código. A ferramenta que pode ser utilizada para o desenvolvimento de códigos em HTML5 e CSS3 é o Visual Studio Code, um editor de código-fonte desenvolvido pela Microsoft.

Obs: as tags em Html5 são as "caixinhas", nas quais temos abertura de tag e fechamento de tag, porém existem tags que fecham e iniciam em si como "hr".
